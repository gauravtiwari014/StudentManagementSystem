# Student Management System

A console-based Student Management System developed using Core Java. The application allows users to manage student records efficiently through CRUD operations.

## Features
- Add Student
- View Students
- Search Student by ID
- Update Student Details
- Delete Student Records
- Menu-Driven Interface

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- ArrayList
- Collections Framework
- Exception Handling

## Project Structure
- Student.java
- StudentManagementSystem.java

## Learning Outcomes
- Implemented OOP concepts such as Encapsulation and Classes.
- Worked with Java Collections Framework.
- Developed CRUD-based functionality.
- Improved problem-solving and Java programming skills.

## How to Run
1. Compile the files:
   javac Student.java StudentManagementSystem.java

2. Run the application:
   java StudentManagementSystem
